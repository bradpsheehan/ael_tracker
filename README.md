# AelTracker

Ruby wrapper for Advanced Energy Legislation Tracker API.

## Installation

Add this line to your application's Gemfile:

    gem 'ael_tracker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ael_tracker
